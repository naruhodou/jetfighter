Jetfighter (Graphics Assignment 2)
=========================

-Controls
    - Forward motion: W
    - Backward motion: S
    - Left turn: A
    - Right turn: D
    - Anti-clockwise tilt: Z
    - Clockwise tilt: C
    - Increase height: SPACE
    - Decrease height: DOWN arrow key
    - Missile shooting: LEFT MOUSE CLICK
    - Bombs: RIGHT MOUSE CLICK
    - Plane view: 1
    - Tower view: 2
    - Follow-cam view: 3
    - Tower view: 4
    - Helicopter view: 5
    - Barrel roll: L
